# RVM Merchant Platform

This is the RVM Merchant Platform - a web application for managing RVM (Reverse Vending Machine) merchants.

## Features
- Dashboard with real-time stats
- User and merchant management
- Submission reviews
- Withdrawal management
- Agent dashboard
- Big data analytics
- Cleaning logs

## Tech Stack
- Vue 3
- TypeScript
- Supabase
- Vercel

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

This project is deployed on Vercel. Push to main branch to trigger deployment.
